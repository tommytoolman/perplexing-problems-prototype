# Original extension — Chapters 15–26

Chapters 15–26 are created for this interactive project. They do **not** appear in Thomas Povey's *Professor Povey's Perplexing Problems*, and their chapter names, problem titles, scenarios, diagrams, interactions and solutions are wholly original.

The extension deliberately begins only after the complete 14-chapter, 109-problem source index. Every extension chapter index and every extension problem page must repeat that boundary prominently.

## Page contract

Every extension page must include this statement or a typographically equivalent version:

> **Original extension.** This chapter and activity were created for this project and do not appear in *Professor Povey's Perplexing Problems*.

Extension pages retain the established challenge, interactive laboratory, progressive hints, revealable solution, accessible diagram, touch-friendly controls and development-only debug gate. Exact results and simulations must be visually distinguished, and every idealisation must be stated.
